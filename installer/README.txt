To install on Windows:
 - Singleplayer
   - make sure you played 1.13.2 at least once
   - double-click on win_install_singleplayer
   - In launcher - add a profile for 1.13.2-carpet, you should be able to see it under available versions
 - Server
   - place 'server.jar' of 1.13.2 downloaded from Mojang in this directory
   - doubleclick on win_install_server
   - it should create patched server file next to it
